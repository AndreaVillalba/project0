BANANA SORTING SOFTWARE

This banana sorting software was created using convolutional neural networks and deep learning techniques.

Overview

This is a software capable to do the distinction of the maturation state of the plantain: ripe, green or damaged, just by uploading a picture to the web app. Our long-term goal is to involve this type of software in industrial processes in order to automate the classification process.

The classification software was created with python, using libraries like Tensorflow and OpenCv, which help us to train the convolutional network. After this was done, we uploaded the trained model to a web application that was developed using Flask, HTML and CSS.

Kickoff

This project started at April of 2019. It was separated into 3 phases, the first one consisted basically of the recolection of images that will be part of the dataset, at the end we were able to have 650 images of each state of maduration, so in total our dataset had almost 2000 images.

The second phase started once the dataset was created, and it consisted in creating a predictive classification program using CNN. To accomplish this, we first had to label the images using numbers, so after this was done we where able to train at least 8 different models so we can find the most accurate one. At the end the most accurate model was the one that performed 2 convolutions, used 256 neurons in each layer and did not use any dense layer in addition to the last one.

As mentioned in the overview, after we found the most accurate model it was saved and connected to the web application. This software allows the user to upload the picture of the plantain of their choice so the software will make its final prediction, if it thinks the platain is: ripe, mature or damaged.

Key challenges

  • One of the things that were really hard at the beggining was to accumulate the correct ammount of pictures, so the model would not memorize but actually learn. It is important to mention that all the pictures that we used, were taken by us, so they had to follow certain conditions such as taking the pictures from various angles, with different types of lighting and with the same background color (white).
  • Each model took a long time to be trained, so getting the one that was really the most efficient took us a long time.

Key results

An accuracy of 96% was achieved, since there were certain classifications that the model could not correctly predict, as shown by the confusion matrix below. It is observed that there were 18 images of green plantain which predicted that they would be ripe. As you can see, the classification of the mature ones was almost perfect, confusing only one with the green one.

In addition this project was finalist of "La Jornada de Iniciación Científica" (Scientific Initiation Contest) that was celebrated nationwide in Panama City.
