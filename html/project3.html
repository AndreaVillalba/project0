SCOOTER MANAGEMENT WEB APP

The purpose of the scooter app is to manage the user authentication as well as the process of renting and returning it back.

Overview

This app is capable of registering new users as well as authenticating the log in of the employees and users. In addition to this, the employee is able to register new stations and scooters, while the user can rent and return a specific scooter as well as registering an incident if necessary. In addition, the user can obtain a list of their previous tours.

Kickoff

This c# application was done using a multilayer arquitecture, where the presentation layer which includes the forms that make up what is the graphical interface. On the other hand, we had the business logic layer, where all the calculus and fuctions were done. At last we created the persistence layer, the one in charge of access to the data.

To facilitate the work we used Entity Framework, a persistence framework developed by Microsoft for the creation of the .net platform, which offers object-relational mapping and a design based on the Repository + Unit of work patterns. In other words, it is constantly looking to the objects and if anything changes it automatically update the database.

Key challenges

The most challenging thing for us was to actually mantain a multilayer arquitecture without compromising the layers in between.
